# Format:
#   - `##`-prefixed lines mark the start of a puzzle; everything
#     after the `##` (and optional space) is the display name.
#   - `#`-prefixed lines (single hash) are free-form comments,
#     ignored by the parser.  Use them for attributions, notes,
#     or dividers.
#   - The nine lines immediately following each `##` name line are
#     the puzzle rows.  Each row is nine characters: digits 1-9
#     for hints, `.` for empty cells.
#   - Blank lines anywhere are ignored.
#
# Thirty-one curated puzzles, ordered from easy to hard.  Timings
# are measured on the reference dev box (aarch64 Apple Silicon via

# latter is brutal for HUMAN solvers (forcing chains) yet merely
# slow for a backtracker; the machine-hard counterpart lives in
# sudoku-stress-tests.txt.  Add your own puzzles by appending
# another `## Name` block.
